## Learning Git and GitHub

Welcome to your first steps with Git and GitHub! Here's a quick overview:

### Key Git Commands
- `git status` - Check what files have changed
- `git add .` - Stage changes for commit
- `git commit -m "message"` - Save your changes with a description
- `git push` - Upload your changes to GitHub
- `git pull` - Download changes from GitHub

### GitHub Workflow
1. Create a **branch** for your changes
2. Make changes and **commit** them
3. Open a **Pull Request (PR)** to propose your changes
4. Review and **merge** when ready
